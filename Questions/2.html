Block Elements

Elements which start from a new line and they take complete width of the screen

Examples

  1. h1 - h6 tags
  2. p tag
  3. div tag

Inline Elements

Elements which start from the point where the previous element ended

Examples

  • span tag
  • video tag
  • audio tag
  • image tag
